## üöÄ Quick Start

### 1. Prerequisites
- Node.js installed.
- A MongoDB Atlas account (for the cloud database).

### 2. Installation
```bash
# Clone the repo
git clone <your-repo-link>

# Setup Backend
cd backend
npm install
# Create a .env file and add your MONGODB_URI

# Setup Frontend
cd ../frontend
npm install
```

### 3. Running Locally
**Start Backend:**
```bash
cd backend
npm run dev
```

**Start Frontend:**
```bash
cd frontend
npm run dev
```
